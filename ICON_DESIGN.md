# LucidFocus Icon Design

## Design Concept

The LucidFocus icon represents **clarity** and **focus** through:

1. **Concentric Focus Rings** - Visual metaphor for focus and concentration
2. **Central Crystal/Diamond** - Represents clarity, lucidity, and precision
3. **Light Rays** - Symbolize illumination and clear vision
4. **Emerald Green Theme** - Matches the brand's color scheme (#10b981)

## Icon Versions

### Version 1: `lucidfocus-icon.svg` (Detailed)
- More complex design with multiple layers
- Rich gradients and depth
- Best for larger sizes (app icons, headers)

### Version 2: `lucidfocus-icon-simple.svg` (Simplified)
- Clean, minimal design
- Better for small sizes (favicons, small logos)
- More scalable and recognizable

## Usage Recommendations

- **App Icon**: Use detailed version (512x512px)
- **Favicon**: Use simple version (32x32px, 64x64px)
- **Header Logo**: Use simple version (48x48px)
- **Social Media**: Use detailed version (1200x1200px)

## Color Palette

- **Primary**: #10b981 (Emerald Green)
- **Secondary**: #34d399 (Emerald Light)
- **Accent**: #059669 (Emerald Dark)
- **Background**: White with 10% opacity green overlay

## Design Principles

1. **Scalability**: Works at any size from 16px to 512px+
2. **Clarity**: Simple enough to be recognizable at small sizes
3. **Brand Alignment**: Matches the "clarity" and "focus" messaging
4. **Modern**: Clean, contemporary design aesthetic
